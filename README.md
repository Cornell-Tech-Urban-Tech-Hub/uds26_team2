# Streets for the People

Jay Huang · Edmond Chak Chun Kong · Nadav Yochman · Nikhil Jain · Ishan Parag Patwardhan

This project studies how parking in the Gowanus BID can transition from car-dominated curb space toward a more walkable, public-realm-focused street system. We combined neighborhood research, GIS data, parking occupancy analysis, scenario modeling, and visual simulation to understand how future growth could affect curb demand.

The site presents five phases of the work: neighborhood insight, site transformation, simulation, overflow prediction, and consensus building. The final consensus section includes an interactive voting prototype where community members can mark parking spots to protect or remove and leave comments, showing how design proposals can be tested with residents, businesses, agencies, and garage developers.

Built as a static GitHub Pages site using HTML, CSS, JavaScript, Leaflet, and Chart.js.
